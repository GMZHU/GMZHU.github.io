自签名证书

局域网／自己服务器搭建自签名证书

生成私钥
$sudo openssl genrsa -des3 -out app.key 1024

自行签署
$sudo openssl req -new -key app.key -out app.csr

服务器私钥
$sudo openssl rsa -in app.key -out server.key

生成服务器签署证书
$sudo openssl req -new -x509 -days 365 -key server.key -out server.crt
